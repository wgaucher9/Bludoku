# Bludoku

A personal iOS Sudoku app built as a Swift/SwiftUI learning project. Bludoku ships with 100 hand-verified puzzles across three difficulty levels and tracks your solve history locally on device.

> **Personal learning project** — built to practice SwiftUI, MVVM architecture, and iOS app development.

## Features

- **100 puzzles** across Easy, Medium, and Hard difficulty levels
- **Difficulty badges** — color-coded (green / orange / red) at a glance
- **Cell highlighting** — selected cell, shared row/column/box, and matching digits all highlighted automatically
- **Input validation** — tap "Check" to highlight incorrect entries in red without revealing answers
- **Digit toggle** — tapping a digit already in a cell erases it
- **Puzzle completion tracking** — solved puzzles are saved to device with a timestamp; a green checkmark appears in the puzzle header for completed puzzles
- **Stats & history view** — bar-chart toolbar button opens a sheet showing total puzzles solved, per-difficulty breakdown, and a full completion history with dates
- **Next-puzzle confirmation** — if you tap Next with unsolved entries, a confirmation dialog prevents accidental progress loss
- **Restart button** — resets the current puzzle to its original state

## How Puzzle Generation Works

The puzzles bundled in `puzzles.json` were generated by a separate Python script (in its own repository). The generator:

1. Builds a valid completed Sudoku grid using backtracking
2. Removes clues one at a time while verifying the puzzle still has a unique solution
3. Classifies the difficulty based on which solving techniques are required (e.g. naked singles → Easy)
4. Outputs a JSON file with each puzzle's clue grid, solution grid, difficulty, and required techniques

The app loads the JSON at launch via `Bundle.main` — no network requests, no server required.

## Setup & Running

**Requirements**
- Xcode 17 or later
- iOS 16.0+ device or simulator

**Steps**

1. Clone the repository:
   ```bash
   git clone https://github.com/wgaucher9/Bludoku.git
   cd Bludoku
   ```

2. Open the project in Xcode:
   ```bash
   open SudokuApp.xcodeproj
   ```

3. Select a simulator or connected device in the scheme selector.

4. Press **Cmd+R** to build and run.

No external dependencies or package manager setup required — the project is pure Swift/SwiftUI with no third-party libraries.
